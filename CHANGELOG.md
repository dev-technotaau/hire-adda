# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
-   **Frontend**: Next.js 14 setup with Tailwind CSS, Shadcn UI.
-   **Backend**: Node.js/Express API with TypeScript.
-   **Auth**: Full JWT authentication flow (Login, Register, MFA, Forgot Password).
-   **Search**: Elasticsearch integration for Jobs and Candidates.
-   **Jobs**: Job posting, editing, and application system.
-   **Profiles**: Comprehensive attributes for Candidates and Companies.
-   **Admin**: Dashboard with analytics, user management, and moderation tools.
-   **Docker**: Production-ready Dockerfile and Compose setup.

### Fixed
-   Resolved middleware redirect loops.
-   Fixed auth token persistence (Cookie/LocalStorage sync).
-   Addressed TypeScript errors in build pipeline.
